# Basic Tracker

A simple, intuitive Android app for tracking habits, health events, and daily activities.

## Features

- **Track anything**: Habits, health events (migraines, mood), activities, and more
- **Sentiment types**: Mark items as positive (good habits), negative (things to avoid), or neutral
- **Multiple views**: Day, Week, and Month views for easy tracking
- **Visual indicators**: Color-coded dots show tracked events at a glance
- **Statistics**: View occurrence counts and patterns over time
- **Privacy first**: All data stored locally, no account needed, no internet required

## Tech Stack

- Kotlin
- Android SDK 35
- SQLite for local storage
- Material Design components

## Building

Open the project in Android Studio and build using Gradle.
